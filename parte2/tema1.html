Códigos para el deletreo de letras y cifras

Alfabeto fonético OACI/OTAN

El alfabeto fonético internacional utilizado en radiocomunicaciones fue adoptado por la Organización de Aviación Civil Internacional (OACI) y la OTAN. Permite deletrear palabras de forma clara y sin ambigüedad, evitando confusiones entre letras que suenan similar (B/V, M/N, etc.).

Código para cifras

Las cifras se pronuncian de forma clara, separando cada dígito:

Cifra Pronunciación
0CERO (ZERO)
1UNO (ONE)
2DOS (TWO)
3TRES (THREE)
4CUATRO (FOUR)
5CINCO (FIVE)
6SEIS (SIX)
7SIETE (SEVEN)
8OCHO (EIGHT)
9NUEVE (NINE / NINER)
Nota sobre "NINER"
En comunicaciones internacionales, el 9 se pronuncia "NINER" para evitar confusión con "FIVE" en inglés.

Carácter Denominación
/ (barra)STROKE o PORTABLE

Uso en radioafición

El alfabeto fonético se utiliza principalmente para:

  • Identificación: Deletrear el indicativo propio y el del corresponsal
  • QTH: Deletrear nombres de localidades o localizadores
  • Nombres: Deletrear el nombre del operador cuando hay dificultades
  • Concursos: Intercambio de información (serial numbers, multiplicadores)
Buena práctica
Siempre que las condiciones de propagación sean difíciles o haya ruido, utilizar el alfabeto fonético para evitar errores. Es obligatorio identificarse con el indicativo completo.
